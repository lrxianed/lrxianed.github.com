插入图片

Aug 24th, 2013 | Comments

如果你喜欢markdown插入图片的语法，那么你也一定会喜欢Octopress简洁的方式，而且附带更多的参数，有更丰富的样式。

语法

｛% img [class names] /path/to/image [width] [height] [title text [alt text]] %｝

一个例子讲解

Include Code 嵌入代码文件

Aug 24th, 2013 | Comments

原来Octopress还可以在文章中直接嵌入代码文件的，好强大的样子。

语法

｛% include_code [title] [lang:language] path/to/file %｝

一个例子讲解

｛% include_code 我的第一JavaScript游戏 lang:js game.js %｝

讲解

  • 默认的代码文件默认的存放文件夹（也就是代码的根目录）是source/downloads/code，如果没有你可以自己创建；如果你想修改默认的存放路径，可用在_config.yml中修改。
  • 如果你的代码存放在source/downloads/code，那么文件名就是路径……不解释，小白都应该懂了；
  • [title]是代码的标题；
  • [lang:language]不是必要的，此插件会自动识别代码的语言分类，但不是所有语言种类都能识别，写上就不会错的啦！

Backtick Code Blocks 代码高亮

Aug 24th, 2013 | Comments

有了此插件，你可以在文章中随意地插入代码，你还可以附带上代码标题以及下载链接。规则很简单，一看就懂。

代码

``` [language] [title] [url] [link text]
    代码区
```

一个例子讲解

``` ruby Discover if a number is prime http://www.noulakaz.net 源码下载
class Fixnum
  def prime?
    ('1' * self) !~ /^1?$|^(11+?)\1+$/
  end
end
```

讲解：

  • [language] [title] [url] [link text] 这些参数是不必要的。不填就没有代码标题、代码高亮等效果；
  • Discover if a number is prime #代码标题；
  • ruby #标记语言；
  • http://www.noulakaz.net #代码下载链接地址；
  • 源码下载 #下载链接的标题；

飞一般的感觉

Aug 18th, 2013 | Comments

感谢国家感谢党和人民！！

今天，Mac mini换上了美光的M500 SSD，速度很喜人，行云流水，打开一个软件只需跳两下，一二，两秒。连开Photoshop这样的大型软件也只是两下，那个爽啊，想开什么立刻就可以打开，不在需要焦急的等待，飞一般的感觉！

凡事有个章可循：

我没有重装系统，我只是把原来SATA上的系统拷贝了一份到SSD中，用的是CCC，是从这篇教程中了解到的：《Macbook Pro 升级 SSD 固态硬盘+数据转移完全教程》。

拆机的教程则是学习了这个视频教程：OWC 介绍 2012 Mac mini 添加第二块硬盘

大家注意啦！！

  • 由于资料还没有整理好就开始拷贝工作，导致有些混乱，如果有什么可以告诫的，就是要先整理好文件，明确哪些资料要搬到ssd，哪些资料留在SATA里。
  • 另外，尽量将没用的不常用的软件都删掉吧，不想删就不要拷贝到ssd中，浪费宝贵的空间。
